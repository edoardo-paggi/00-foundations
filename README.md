# 00 - Foundations

This section contains the core foundations of my Computer Science journey.

The goal is to build a strong understanding of programming fundamentals, development tools, operating systems, and low-level computer behavior before moving to advanced Computer Science topics.

## Goals

- [ ] Learn the C programming language
- [ ] Understand variables, functions, arrays, structs, and pointers
- [ ] Learn dynamic memory allocation and manual memory management
- [ ] Understand how compilation and linking work
- [ ] Learn debugging and problem solving
- [ ] Become comfortable with the Linux terminal and shell commands
- [ ] Understand basic operating system concepts and processes
- [ ] Learn Git and GitHub workflow
- [ ] Build small command-line projects in C
- [ ] Write clean and organized code
