# Chapter 3 – Formatted Input/Output

This chapter introduces formatted input and output in C through the `printf`
and `scanf` functions. It explains how conversion specifications such as `%d`,
`%f`, `%e`, and `%g` are used to control the representation of data, along with 
field width, precision, and escape sequences. The chapter also covers the 
correct use of `scanf` for reading user input and highlights common mistakes 
that can lead to unexpected behavior.
