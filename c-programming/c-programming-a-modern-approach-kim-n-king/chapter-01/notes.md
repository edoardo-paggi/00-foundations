# Notes - Chapter 01

The C programming language was born as a byproduct of the UNIX operating system.

It evolved from a small language called B.
B was influenced by BCPL, which was influenced by CPL.
CPL was heavily influenced by ALGOL.
ALGOL was influenced by Fortran, and Fortran by Speedcoding.
Finally, Speedcoding was influenced by Assembly and Machine Language.

- Machine Language
- Assembly
- 1953: Speedcoding
- 1957: Fortran
- 1958: ALGOL
- 1963: CPL
- 1966: BCPL
- 1969: B
- 1972: C

## Languages Influenced by C

- 1983: C++
- 1987: Perl
- 1995: Java
- 2000: C#

## Key Concepts

C was originally intended for system programming and machines with limited memory,
although today it is used in many different fields such as embedded systems,
GUI applications, operating systems, and computer graphics.

### Strengths

- Efficiency and low memory usage
- Portability
- Power
- Flexibility
- Originally designed for system programming
- Rich standard library
- Strong integration with UNIX and Linux systems

### Weaknesses

- Programs can contain many errors
- Programs can become difficult to understand
- Programs can become difficult to maintain and modify
