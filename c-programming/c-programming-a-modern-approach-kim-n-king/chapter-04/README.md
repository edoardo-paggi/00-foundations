# Chapter 4 – Expressions

Chapter 4 introduces expressions, the fundamental building blocks used to 
perform computations and manipulate data in C. It explains how arithmetic 
operators are used to combine values and produce results, while assignment 
operators store those results in variables and support concise update 
operations. The chapter also covers increment and decrement operators, which 
provide a compact way to increase or decrease a variable's value by one. 
A significant focus is placed on expression evaluation, including operator 
precedence, associativity, and the order in which subexpressions are processed 
to determine a final result. Finally, it introduces expression statements, 
showing how expressions can be used as complete executable statements within a 
program, particularly when performing assignments, updates, or function calls.
