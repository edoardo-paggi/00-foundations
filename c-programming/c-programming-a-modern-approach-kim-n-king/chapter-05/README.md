# Chapter 5 – Selection Statement

Chapter 5 introduces C's decision-making constructs, which allow programs to 
execute different actions based on conditions. Logical expressions combine 
relational and logical operators to produce true or false results, forming the 
foundation of program control flow. The `if` statement enables conditional 
execution of code blocks, supporting simple decisions as well as more complex 
branching through `else` and nested conditions. The `switch` statement provides 
an alternative to long chains of if statements when selecting among multiple 
constant values, using `case` labels and optional `default` handling to direct 
program execution efficiently.
