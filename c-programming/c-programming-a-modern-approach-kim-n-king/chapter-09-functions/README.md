# Chapter 8 – Functions

This chapter introduces functions, one of the fundamental building blocks of C 
programs. Functions allow code to be organized into reusable, modular units, 
making programs easier to understand, maintain, and extend. The chapter explains 
how to define and call functions, how function declarations enable separate 
compilation, and how arguments and return values are used to exchange data 
between functions. It also discusses program termination and the role of the 
`return` statement in controlling both function execution and the program's exit 
status. Finally, the chapter introduces recursion, a programming technique in 
which a function calls itself to solve a problem by breaking it down into 
smaller subproblems.
