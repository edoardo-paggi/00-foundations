# Chapter 6 – Loops

This chapter introduces iteration in C, one of the fundamental techniques used 
to control program flow. It covers the three loop constructs provided by the 
language: `while`, `do-while`, and `for`, explaining their syntax, behavior, and
typical use cases. The chapter also discusses how loops can be terminated before
their normal completion, allowing programs to respond to specific conditions 
during execution. Finally, it introduces the null statement and demonstrates 
how it can be used in situations where a loop or control structure requires a 
statement but no action needs to be performed.
