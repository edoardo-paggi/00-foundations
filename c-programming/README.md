# C Programming

This section focuses on learning the C programming language and understanding low-level programming concepts.

The main goal is not only to learn C syntax, but also to develop a deeper understanding of how computers work under the hood, including memory management, compilation, debugging, and data representation.

## Goals

- [ ] Learn C syntax and core language features
- [ ] Understand pointers and memory management
- [ ] Work with arrays, strings, structs, and files
- [ ] Learn dynamic memory allocation
- [ ] Understand compilation and linking
- [ ] Practice debugging and problem solving
- [ ] Write clean and modular C code
- [ ] Build small command-line applications
- [ ] Reimplement parts of the C standard library
- [ ] Develop low-level programming intuition
